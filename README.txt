Sofa Online - Minimal PHP + MySQL system with antar-jemput features
Instructions:
1. Place folder in PHP webroot (e.g., /var/www/html/) or run with built-in PHP server:
   php -S localhost:8000 -t /path/to/sofa_online
2. Import database: use database.sql into MySQL (create database 'sofa_online').
3. Edit config.php for DB credentials.
